## 📘 Project Title

AI Image Generator App

## 📝 Description

This app lets you type any idea into a text box, and it uses AI to generate an image based on your words. It sends your prompt to an AI image model and shows the created picture on your screen.

The app works on iOS and Android (not on web) and is simple and beginner-friendly.

## ⭐ Basic Features (Explained Simply)

* **Text Prompt Box:** You type what you want the AI to draw.
* **Generate Button:** Sends your text to the AI to create an image.
* **Loading Indicator:** Shows a spinner while the app waits for the AI to finish.
* **Displays the Image:** If the AI returns a picture, it appears on your screen.
* **Error Handling:** Shows a message if something goes wrong (empty prompt, bad API key, etc.).bash npx expo start -c --tunnel

##  How to Run the App

1. **Download the project**
   Click the **green "Code" button** on GitHub → **Download ZIP**.
   Extract the folder.

2. **Open the project in VSCode**

3. **Open the terminal and run:**

   ```bash
   npx create-expo-app@latest ./
   ```

4. **Start the app (tunnel mode):**  



   ```bash
   npx expo start -c --tunnel
   ```

5. **Scan the QR code** using the **Expo Go** app on your mobile device to see the app running.
